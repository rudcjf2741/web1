  • 배운태그 : strong, u, h1~h6

    • strong : 글자를 진하게 해준다
    예시 : 가나다라마바사

    u : 글자에 밑줄을 쳐 준다
    예시 : 가나다라마바사

    h1~h6 : 제목별로 1~6까지의 중요순서?크기순서를 나타낸다.
    예시 :

    3일차

  • 배운태그 : br, p, img

    • br : 문장의 줄바꿈을 해 준다.
    예시 : 안녕하십니까 저는 정보통신공학과 20171075 김경철입니다.
    지금부터 발표를 시작하도록 하겠습니다.

    p : 문단(단락)을 나누어 준다.
    예시 :

    안녕하십니까 ..... 하겠습니다.

    저는 어릴때 부터...하겠습니다.

    마지막으로....감사합니다.
